📐 新旧风格对比

旧风格 SWEETIE-16

草莓 32×32

特点：无轮廓线 / 糖果色调 / 可爱卡通风

新风格 HARVEST-20

萝卜 16×16
苹果 16×16
鸡蛋 16×16
特点：1px暗色轮廓 / 左上高光 / 温暖泥土色
独有：魔法微光点 / 可动画 / 限定20色调色板

📋 风格规则速查

  • 轮廓线：所有物品必须有 1px #1a1c2c 轮廓，是物品最外层像素
  • 光源：固定左上 45°，高光在左上 1-2px，暗影在右下 1-2px
  • 尺寸：物品/道具 16×16，角色/场景 32×32
  • 魔法微光：每个物品有 1px #f4f4f4 或 #73eff7 高光点（我们的视觉签名）
  • 造型：偏圆润，比 SV 略萌，食物/果实有膨胀感
  • 调色板：严格限定 HARVEST-20，不引入调色板外的颜色
