# 风格指南 — Cozy Pixel (舒适像素)

## 核心理念

星露谷的温暖田园感 + 我们自己的魔法微光。
不是 SV 的复制品，是"如果星露谷物品有了生命和魔力"。

## 绘制规则

### 1. 轮廓线
- **所有物品必须有 1px #1a1c2c 轮廓线**
- 轮廓线是物品最外层像素，不是额外添加的边框
- 角色轮廓可用 #333c57（稍柔和）

### 2. 光影
- **光源：左上方 45°**
- 每个色相使用 2-3 个梯度：高光 / 主色 / 暗面
- 高光在左上 1-2px
- 暗影在右下 1-2px
- 可用 #f4f4f4 做 1px 镜面高光点（"魔法微光"，这是我们的标志）

### 3. 尺寸
- 物品/道具：**16×16**
- 角色/场景：**32×32**
- 所有图标居中放置，边缘留 1-2px 呼吸空间

### 4. 造型
- **偏圆润**，避免尖锐直角（除非物品本身是方形的）
- 比 SV 略萌一些——这是我们和 SV 的区别
- 食物/果实略微膨胀感（可爱感的来源）

## 题材方向

### 优先（星露谷核心循环）
- 🌾 农作物（成长阶段）
- 🔧 工具（锄头、水壶、斧头、镐、钓竿）
- 🐔 农场动物
- 🐟 鱼类
- 💎 矿石
- 🍳 料理
- 🌸 四季物品

### 保留（已有且合适的）
- 自然场景、花草植物、天气
- 可爱动物、食物饮品

### 淡化（不太匹配新方向的）
- 世界文化符号、幻想生物过多、过于现代的物品

## "我们的风格" vs 星露谷

| SV | 我们 |
|-----|------|
| 写实田园 | 微魔法田园 |
| 物品是实用工具 | 物品像是有灵魂的 |
| 无动画（静态精灵图） | **每个物品都能动**（核心差异） |
| 数百种颜色 | **HARVEST-20 限定调色板** |
| 纯游戏素材 | SVG 矢量，任意尺寸清晰 |
