## 📦 Usage · 使用

```html
<img src="assets/heart.svg" width="64" style="image-rendering: pixelated" alt="Pixel Heart">
```

> **Tip:** Always add `image-rendering: pixelated;` in CSS to keep sharp pixel edges when scaling up.
>
> **提示：** 放大显示时务必添加 `image-rendering: pixelated;`，保持像素边缘锐利。

```css
img.pixel-art {
  image-rendering: pixelated;
  image-rendering: crisp-edges; /* Firefox fallback */
}
```

```bash
git clone https://github.com/Furinaaa-Cancan/pixel-art.git
```
