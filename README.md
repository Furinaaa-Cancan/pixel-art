## 功能特性 / Features

| 功能 | Feature | 说明 / Description |
|------|---------|-------------------|
| 难度选择 | Difficulty Selection | 简单 / 普通 / 困难三档，影响速度和加速率 — Easy / Normal / Hard, affecting speed and acceleration |
| 触屏支持 | Touch Support | 滑动控制方向 + 虚拟方向键，移动端自动适配 — Swipe to control + virtual D-pad, auto-detected on touch devices |
| 暂停功能 | Pause | 空格键暂停/继续，触屏点击画布暂停 — Press Space to pause/resume, or tap the canvas on mobile |
| 最高分记录 | High Score | 使用 localStorage 持久化保存历史最高分 — Persisted via localStorage across sessions |
| 音效系统 | Sound Effects | 基于 Web Audio API，吃食物和死亡时播放音效 — Web Audio API powered, plays on eat and death events |
| 粒子特效 | Particle Effects | 吃食物时绿色粒子爆发，死亡时红色粒子爆发 — Green burst on eating, red burst on death |
| 视觉升级 | Enhanced Visuals | 蛇身圆角渐变、蛇头发光眼睛、食物脉冲闪烁 — Rounded gradient body, glowing head with eyes, pulsing food |
